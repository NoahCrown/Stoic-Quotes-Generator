# Stoic Quotes Generator
This is a Stoic Quotes Generator web application built with HTML, CSS, JavaScript, React, and a Stoic API. The app generates quotes, sayings, and insights from famous Stoic philosophers such as Seneca, Marcus Aurelius, and Epictetus.

![stoic-quotes-generator netlify app_](https://user-images.githubusercontent.com/91674419/205507414-d5689dbd-ab47-4273-b135-a6bbe8eb026f.png)

# Features
* Daily Stoic quote generator
* Quote sharing on social media
* Modern and sleek design

# Installation
To install this application, you must first clone this repository to your local machine. Then, navigate to the project directory and run the following command:
npm install
This will install all the necessary dependencies required for the application to run.

# Usage
To start the application, run the following command:
npm start
This will start the application on your local server. You can then access the application by navigating to http://localhost:3000 in your web browser.

# API
This application also utilizes a Stoic API to generate quotes. To use the API, you will need to sign up for an API key at https://stoicquotesapi.com/.
